В примере, рассмотренном в начале главы, было показано, как создать одну структуру, хранящую информацию о студенте. Представим, что нам нужно хранить несколько структур в памяти - для нескольких объектов. Когда мы знаем количество объектов, мы может объявить [[Что такое массив|массив]] структур:
```cpp
struct student
{
	char name[50];
	int roll;
	float mark;
} s[10];
```

`struct student s[10]` объявляет массив из 10 элементов типа `student`. Каждый элемент массива является структурой `student`, которая содержит три поля: `name` (строка из 50 символов), `roll` (целое число) и `mark` (число с плавающей точкой).

Таким образом, `s[10]` представляет собой массив из 10 структур `student`. Этот массив может содержать данные о 10 студентах. каждый элемент массива `s` будет представлять отдельного студента, и у каждого студента будет имя, номер зачетки и оценка.
