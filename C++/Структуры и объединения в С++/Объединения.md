В C++, объединение (union) - это тип данных, который позволяет объединять несколько переменных различных типов в одной области памяти. В отличие от структур, где каждый элемент имеет свою собственную область памяти, элементы объединения используют одну и ту же область памяти.

Основные особенности объединения:

1. **Одинаковое расположение в памяти**: Все элементы объединения разделяют одну и ту же область памяти. Это означает, что изменение одного элемента может повлиять на другие элементы в этом объединении.

2. **Размер равен размеру наибольшего элемента**: Размер объединения равен размеру его наибольшего элемента, так как все элементы используют одну и ту же область памяти.

3. **Только одно поле может быть активным**: В любой момент времени только одно поле объединения может содержать данные. Обращение к неактивным полям может привести к неопределенному поведению.

4. **Удобство при работе с байтами**: Объединения могут быть полезны при работе с байтами в памяти, так как они позволяют интерпретировать область памяти разными способами.

Пример объявления и использования объединения в C++:

```cpp
#include <iostream>
using namespace std;

union MyUnion {
    int i;
    float f;
    char c;
};

int main() {
    MyUnion u;
    u.i = 10;
    cout << "Integer: " << u.i << endl;  // Вывод: Integer: 10
    cout << "Float: " << u.f << endl;    // Неопределенное поведение, так как float неинициализирован
    cout << "Character: " << u.c << endl;  // Неопределенное поведение, так как char неинициализирован
    return 0;
}
```

В этом примере `MyUnion` объединяет три элемента: целочисленное (`int`), число с плавающей запятой (`float`) и символьное (`char`). Обратите внимание, что при присвоении значения одному из элементов, другие могут содержать мусорные данные.
