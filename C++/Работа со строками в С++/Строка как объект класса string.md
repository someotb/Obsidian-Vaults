Строка как объект класса `string` может быть определена гораздо проще:
`string имя_переменной;`

Например:
`string str;`
Только при этом необходимо подключить заголовочный файл `<string>`.

При задании строки-объекта ее можно сразу инициализировать, то есть присвоить значение по умолчанию. Например:
	`string str = "Hello, world!";`


В C++ существует множество функций и методов, которые могут работать со строками типа `std::string`. Некоторые из них входят в стандартную библиотеку C++, а некоторые могут быть определены в других библиотеках или пользовательских классах. Вот несколько часто используемых функций и методов для работы со строками типа `std::string`:

1. **`std::getline()`**: Функция для считывания строки из входного потока (например, `std::cin`) в объект типа `std::string`.

```cpp
#include <iostream>
#include <string>

int main() {
    std::string line;
    std::getline(std::cin, line);
    std::cout << "Вы ввели: " << line << std::endl;
    return 0;
}
```

2. **`std::string::size()`**: Метод, возвращающий размер строки.

```cpp
#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, world!";
    std::cout << "Длина строки: " << str.size() << std::endl;
    return 0;
}
```

3. **`std::string::append()`**: Метод для добавления строки к другой строке.

```cpp
#include <iostream>
#include <string>

int main() {
    std::string str1 = "Hello, ";
    std::string str2 = "world!";
    str1.append(str2);
    std::cout << str1 << std::endl; // Выведет "Hello, world!"
    return 0;
}
```

4. **`std::string::substr()`**: Метод для извлечения подстроки из строки.

```cpp
#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, world!";
    std::string sub = str.substr(7, 5); // Извлекаем подстроку, начиная с индекса 7 длиной 5 символов
    std::cout << sub << std::endl; // Выведет "world"
    return 0;
}
```

Это только некоторые из функций и методов, которые могут работать со строками типа `std::string` в C++. Другие операции включают поиск подстрок, замену символов, сравнение строк и многое другое.

Есть несколько способов добавления символов или строк в конец строки типа `std::string` в C++.

1. **Оператор `+=`**: Вы можете использовать оператор `+=`, который добавляет строку или символ в конец строки.

```cpp
#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, ";
    str += "world!"; // Добавляем строку в конец
    std::cout << str << std::endl; // Выведет "Hello, world!"
    
    str += '!'; // Добавляем символ в конец
    std::cout << str << std::endl; // Выведет "Hello, world!!"

    return 0;
}
```

2. **Метод `append()`**: Метод `append()` также может быть использован для добавления строк или символов в конец строки.

```cpp
#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, ";
    str.append("world!"); // Добавляем строку в конец
    std::cout << str << std::endl; // Выведет "Hello, world!"
    
    str.append(3, '!'); // Добавляем три символа '!' в конец
    std::cout << str << std::endl; // Выведет "Hello, world!!!"

    return 0;
}
```

Оба этих метода позволяют добавлять строки или символы в конец строки `std::string` в C++.

Функция `isdigit()` является одной из функций, предоставляемых стандартной библиотекой С/С++ `<cctype>`. Она используется для определения, является ли переданный символ цифрой (от '0' до '9').

Прототип функции `isdigit()` выглядит следующим образом:

```cpp
int isdigit(int c);
```

Функция `isdigit()` принимает один аргумент типа `int`, который должен быть символом (представленным в виде числа) в ASCII-кодировке. Она возвращает ненулевое значение, если переданный символ является цифрой, и 0 в противном случае.

Вот пример использования функции `isdigit()`:

```cpp
#include <iostream>
#include <cctype>

int main() {
    char ch = '7';
    
    if (isdigit(ch)) {
        std::cout << ch << " является цифрой." << std::endl;
    } else {
        std::cout << ch << " не является цифрой." << std::endl;
    }

    return 0;
}
```

В этом примере функция `isdigit()` проверяет, является ли символ `'7'` цифрой. Поскольку это действительно цифра, программа выведет сообщение о том, что символ является цифрой.

Функция `isdigit()` часто используется вместе с другими функциями библиотеки `<cctype>`, такими как `isalpha()`, `isalnum()` и `isspace()`, для проверки различных категорий символов.
