==Константы== - это те же переменные, но только с фиксированным, определенным единожды, значением.

Для этого используется идентификатор const.
Пример:
`const int m = 5;`

Так, отдельные символы - литералы типа char указываются в одинарных кавычках, например 'a' или 'd'.
Для литералов типа wchar_t перед непосредственно значением указывается префикс L. Примеры литералов типа wchar_t: L'a', L'A', L'd', L'D'.
В С++ существуют литералы типа текстовой строки. Соответствующие значения указываются в двойных кавычках. Примером текстового литерала, например может быть фраза "Hello, World!". 

В С++ есть несколько числовых типов, поэтому не всегда очевидно, к какому именно числовому типу относится литерал. Здесь действует общее правило: ==**литерал интерпретируется в пределах минимально необходимого типа.**==
Примеры:
(например, 5.3F) - принадлежность литерала к числу с плавающей точкой.
(например, 5.3L) - принадлежность к типу long double
(например, 122L) - этот же суффикс, но у целочисленного литерала означает принадлежность последнего к типу long int.
(например, 123U) - используется с целочисленными литералами для отнесения их к типу unsigned int.
