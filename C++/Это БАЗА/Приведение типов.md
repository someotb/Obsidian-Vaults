В качестве примера рассмотрим программу, которая использует приведение типов для нахождения ASCII- кода символа, введённого пользователем.
```
#include <iostream>
using namescpace std;

int main()
{
char c;
cout << "Введите символ: ";
cin >> c;
cout << "ASCII-значение: " << int(c) << endl;
return 0;
}
```
