Среднеквадратическое отклонение (standard deviation) - это мера разброса или изменчивости значений в наборе данных. Оно показывает, насколько сильно значения отклоняются от среднего значения.

Вычисление среднеквадратического отклонения происходит в несколько шагов:
1. Найдите среднее значение (арифметическое среднее) набора данных.
2. Для каждого значения в наборе данных вычислите квадрат разницы между этим значением и средним значением.
3. Найдите среднее значение квадратов разностей (это называется дисперсией).
4. Извлеките квадратный корень из дисперсии.
