==Массив== - это множество значений одного типа, объединённых общим именем.

Переменные, входящие в состав массива, называются элементами массива. Доступ к элементам массива осуществляется путем индексирования. Размерность массива определяется количеством индексов, необходимых для однозначного определения элемента массива. 

Для создания массива используется оператор объявления. При этом объявление массива должно задавать три аспекта:
- Имя массива
- Количество элементов массива
- Тип всех элементов массива
