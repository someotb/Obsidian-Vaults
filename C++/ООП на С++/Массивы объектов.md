Объекты - это обычные переменные. Следовательно, мы можем создать массив таких переменных, то есть массив объектов. В примере ниже, создан демо-класс и массив объектов этого класса. Заодно этот пример показывает, как описываются простые функции класса. Также обратите внимание на то, как вызываются функции-члены класса для каждого элемента массива: имя массива индексируется, затем к члену применяется оператор доступа, за которым следует имя вызываемой функции-члена класса.

```cpp
#include <iostream>
using namespace std;

class demo {
	int a;
public:
	void set_a(int n) { a = n; }
	int get_a() { return a; }
};

int main()
{
	demo ar[3];
	int i;
	
	for (i=0; i<3; i++) ar[i].set_a(i);
	for (i=0; i<3; i++) cout << ar[i].get_a() << " ";
	cout << endl;
	return 0;
}
```

Теперь усложним задачу. Представим, что наш demo-класс использует конструктор с параметром. Как тогда инициализировать массив? В этом случае значения, которые будут переданы конструктором, указываются в фигурных скобках

```cpp
#include <iostream>

using namespace std;

class demo {
	int a;
public:
	demo(int n) {a = n;}
	int get_a() {return a;}
};

int main()
{
	demo ar[3] = {3,2,1};
	int i;

	for (i = 0; i < 3; i++) cout << ar[i].get_a() << " ";
	cout << endl;

	return 0;
}
```

Данная программа выведет 3 2 1 - именно эти числа мы передавали при объявлении массива. Вот вывод в терминале:
