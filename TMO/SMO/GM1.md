# Система массового обслуживания G/M/1

## Описание
Система с:
- **G** — произвольное распределение поступления
- **M** — [[Экспоненциальное распределение |Экспоненциальное время]] обслуживания
- **1** — один канал обслуживания

**Параметры:**
- $t_a$ — произвольное распределение интервалов между поступлениями
- $\mu$ — интенсивность обслуживания

## MATLAB код

```matlab
GM1_tn = sort(cumsum(normrnd(1/lambda, 0.2, 1, N))); % случайное поступление
GM1_vn = sort(cumsum(exprnd(1/u, 1, N))); % экспоненциальное обслуживание

figure;
plot(GM1_tn, 0:1:length(GM1_tn)-1);
hold on;
plot(GM1_vn, 0:1:length(GM1_vn)-1);
title("G/M/1: Зависимость числа пришедших/обслуженных заявок от времени");
xlabel("Время,с");
ylabel("Кол-во заявок,шт");
legend("Пришедшие заявки", "Обслуженные заявки");
grid on;
hold off;
