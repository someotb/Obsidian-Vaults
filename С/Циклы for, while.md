### while:
  - Кол-во итераций в while изначально не известно

```c
int i = 0;
while(i < 10)
{
	printf("%d", i);
	++i;
}
```

### for
 - Является "синтаксическим сахаром" не нужен для работы языка C, но удобен в использовании
```c
for (int i = 0; i < 10; ++i)
{
	printf("%d", i);
}

// int i = 0; - инициализация, соответсвует первой строке из цикла while
// i < 10; - условие вхождения, соответствует while(i < 10)
// ++i; - то же что и в while
// цикл for, также можно зациклить, например:
for (int i = 10; i != 9; ++i)
{
	printf("%d", i);
}

// ИЛИ

for (undigned int i = 10; i >= 0; --i)
{
	printf("%d", i);
}

// Если i является беззнаковой переменной, то оно никогда не станет отрицательным, -> условие никогда не будет выполнено -> бесконечный цикл
```
