В языке С, преобразовние типов происходит неявно. Пример:
```c
#include<stdio.h>

int main()
{
    // int неявно приводится к insigned int
    int x = -100;
    unsigned int y = 10U; // 10U - 10 только без знака
    long long int z = x + y;
    printf("z = %lld\n", z);
    return 0;
}

/*
Вывод:
z = 4294967206
*/
```

Пример явного приведения типов:
```c
#include<stdio.h>

int main()
{
    // char явно приводится к типу int
    char c = 'Я';
    int d = (int)c * 10;
    printf("d = %d\n", d);
    return 0;
}

/*
Вывод:
d = -810
*/
```

## Вывод:
Неявное приведение типов происходит всегда, когда типы переменных не соответствуют друг другу, а явное приведение типо выполняется с помощью скобок "(type)" e.g. (int)a.
