Этапы:
- Preprocessing - выполняется препроцессором компилятора, смысл в том, чтобы:
	- удалить комментарии
	- выполнить текстовые замены(макросы: define)
	- включения текста(какой-то текст включаем, какой-то нет)
	- подключение библиотек.
- Compilation - компиляция(трансляция), переводится из языка высокого уровня на язык машинных кодов, после этой обработки появляется объектный файл, который содержит по большей части машинный код, но хранит некоторые символы, например, названия функций
- Linking - компановка(сборка), важная часть компиляции, которая собирает исполняемый файл

```c
#include<stdio.h>

int main(int argc, char* argv[])
```
