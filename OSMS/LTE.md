# LTE — что это такое

**LTE (Long Term Evolution)** — это стандарт мобильной связи четвертого поколения (4G), обеспечивающий высокоскоростную передачу данных для мобильных устройств.

Основные особенности:
- Высокие скорости передачи данных: до нескольких сотен Мбит/с.
- Минимальная задержка (латентность) — важна для приложений реального времени.
- Поддержка IP-трафика для голоса, видео и интернета.
- Совместимость с предыдущими сетями 2G/3G, что позволяет плавный переход.

## Технологические аспекты

- **Модуляция:** OFDMA на канале передачи данных вниз, SC-FDMA на канале передачи данных вверх.
- **Архитектура:** Evolved Packet Core (EPC) обеспечивает маршрутизацию только пакетного IP-трафика.
- **Частоты:** Используются различные диапазоны, в зависимости от страны и оператора.

### Основная формула для пропускной способности канала (теоретическая)

$$
C = B \cdot \log_2(1 + SNR)
$$

где:
- C — максимальная пропускная способность канала (бит/с),
- B — ширина полосы канала (Гц),
- SNR — отношение сигнал/шум.

## Применение

- Высокоскоростной мобильный интернет.
- Передача видео высокой четкости.
- VoLTE (Voice over LTE) для качественных голосовых вызовов.
- Подключение IoT-устройств и мобильных приложений.

## Короткое определение для заметки

> **LTE** — это стандарт мобильной связи 4G, обеспечивающий высокоскоростную передачу данных и низкую задержку для современных мобильных сервисов.
