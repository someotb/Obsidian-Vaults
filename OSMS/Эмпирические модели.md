# Эмпирические модели — что это такое

**Эмпирическая модель** — это математическое или статистическое описание явления, **основанное на наблюдениях и экспериментальных данных**, а не на теоретических законах природы.

Другими словами, это модель, которая **описывает поведение системы на основе опыта**, даже если точный механизм неизвестен.


## Основная идея

> **Эмпирическая модель** — это попытка предсказать результат, не зная всех физических законов, но имея достаточно измерений, чтобы уловить закономерность.


## Примеры

- Прогноз погоды (модели на основе статистики наблюдений).
- Модели роста населения.
- Радиомодели распространения сигнала (например, [[Модель Окумура-Хата и ее модификация COST231|Hata, Okumura, COST-231]]).
- Экономические модели зависимости спроса и цены.


## Общая структура эмпирической модели

Типичная форма:

$$\Huge y = f(x_1, x_2, ..., x_n; a_1, a_2, ..., a_m)$$

где:
-  $x_i$ — входные параметры (независимые переменные),
-  $y$— измеряемый результат,
- $a_j$ — коэффициенты, определённые из эксперимента,
-  $f(\cdot)$ — выбранная зависимость (линейная, степенная, логарифмическая и т.д.).


## Отличие от теоретических моделей

| Критерий | Эмпирическая модель            | Теоретическая модель                         |
| -------- | ------------------------------ | -------------------------------------------- |
| Основа   | Экспериментальные данные       | Физические законы                            |
| Точность | Хорошая в известных диапазонах | Универсальна, но требует допущений           |
| Гибкость | Простая настройка по данным    | Трудно адаптировать без новых уравнений      |
| Пример   | Модель Хата для радиосвязи     | [[Уравнения Максвелла\|Уравнения Максвелла]] |

## Преимущества

- Простота и практичность.
- Высокая точность в пределах измеренных данных.
- Быстрая калибровка под новые условия.

## Недостатки

- Не объясняет физическую природу процесса.
- Может быть неточной при выходе за пределы экспериментальных условий.
- Требует большого массива данных для построения.

## Типы эмпирических моделей

1. **Регрессионные модели** — поиск зависимостей вида y = f(x) по данным.
2. **Модели машинного обучения** — нелинейные эмпирические аппроксимации без явных уравнений.
3. **Феноменологические модели** — упрощённые описания с частично известной физикой.

## Использование в радиосвязи

В радиотехнике эмпирические модели применяются для расчёта потерь распространения сигнала:

$$\Huge PL_{дБ} = A + B\log_{10}(d) + C$$

Пример: модель **Hata** (городская застройка) вычисляет затухание на основе измеренных характеристик, без прямого решения уравнений Максвелла.

## Короткое определение для заметки

> **Эмпирическая модель** — это модель, выведенная из опыта и данных, описывающая наблюдаемое поведение системы без полного знания её внутренней природы.
