Прямоугольный импульс - это функция равная 1 на определенном участке времени и равная 0 вне его:
$$
\Huge
x(t) =
\begin{equation}
\left\{
\begin{aligned}
1, |t|\leq \frac{\tau}{2} \\ 0, |t| > \frac{\tau}{2}
\end{aligned}
\right.
\end{equation}
$$
- где $\Huge \tau$ — длительность импульса.

Пример кода на python:
```python
from matplotlib import pyplot as plt
import numpy as np

tau = 1.0 # Длительность импульса
N = 1000
t = np.linspace(0, 3/tau, N)

x = np.zeros_like(t)
x[np.abs(t) <= tau / 2] = 1.0

plt.plot(t, x)
plt.show()
```

Затем можно сделать спектральный график этого сигнала, для этого нужно выполничть численое интегрирование, в python это можно сделать используя библиотек NumPy и функцию оттуда sum(). Вот пример кода:
```python
from matplotlib import pyplot as plt
import numpy as np

tau = 1
N = 1000
t = np.linspace(-2 * tau, 2 * tau, N)
dt = t[1] - t[0]
x = np.zeros_like(t)
x[np.abs(t) <= tau / 2] = 1.0

plt.plot(t, x)
plt.show()

freq = np.linspace(0, 3 / tau, N)
res = []
for f in freq:
    s = np.sum(x * np.exp(-2j * np.pi * f * t)) * dt
    res.append(s)

res = np.array(res)
ampl = np.abs(res)
phase = np.angle(res)

plt.subplot(2,1,1)
plt.plot(freq, ampl)
plt.xlabel("Частота")
plt.ylabel("Амлитуда")
plt.grid(True)
plt.subplot(2,1,2)
plt.plot(freq, phase)
plt.xlabel("Частота")
plt.ylabel("Фаза")
plt.grid(True)
plt.show()
```
